# CRLF TO LF

CLI to convert CRLF (M$ Win) to LF (Unix / Linux).

## Build

> Requirement: [Rust](http://www.rust-lang.org/), [Cargo](https://crates.io/).

```sh
$ cargo build
```

## Usage

The executable binary file will be built under `target/crlf2lf`.

```sh
Usage: crlf2lf [options] <DIR>
       crlf2lf --help

Options:
  -r --recur
  -h --help
```
